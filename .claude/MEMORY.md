## How to Use

- **Read** a file when its topic is relevant to the current task.
- **Update** an existing file rather than creating a duplicate.
- **Add a new row** to this index whenever a new memory file is created.
- **Suggest updates** when repeated corrections happen or new decisions are made.
- If a decision in `decisions.md` is resolved → move it from "Open" to "Confirmed" with the date.
